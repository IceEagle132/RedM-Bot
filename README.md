# RedM Discord Bot

This bot was made for a Syn County Ranch

**Supported Hosting Options**
- **VPS (Virtual Private Server)** ✅
- **Home Hosted** ✅
---

## Features
- Reads Ranch Webhooks
- Announces when a player collects milk/eggs - buys/sells cattle
- Payout and Wipe commands
